# overtone-template
A Leiningen template for Overtone.

## Usage
--

### Installation

```
$ git clone https://github.com/naoji/overtone-template.git
$ cd overtone-template
$ lein install
```

### Create a project

```
$ lein new overtone overtone-project
```
